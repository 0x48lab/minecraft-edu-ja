# Minecraft Education Edition - 日本語翻訳

Minecraft Education Editionのレッスンで使用されるMakeCodeチュートリアルとカスタムブロックの日本語翻訳です。

## 構成

```
├── ai-pattern-recognition/   # AI パターン認識レッスン
│   ├── PatternRecognition_ACT1.md  # アクティビティ1〜4
│   ├── ...
│   └── pxt.json
├── custom-blocks/            # カスタムブロック定義（日本語）
│   ├── AIBlocks.ts
│   └── pxt.json
└── sources.json              # 元リポジトリとの対応表
```

## 使い方

翻訳済み`.mcworld`ファイルでは、`codebuilder navigate`コマンドのURLが以下のように書き換えられています：

**元:**
```
#tutorial:github:fountainstudios/aimakecodemarkdown/PatternRecognition_ACT2
```

**翻訳後:**
```
#tutorial:github:0x48lab/minecraft-edu-ja/ai-pattern-recognition/PatternRecognition_ACT2
```

## 元リポジトリ

- チュートリアル: [fountainstudios/aimakecodemarkdown](https://github.com/fountainstudios/aimakecodemarkdown)
- カスタムブロック: [fountainstudios/AICustomBlocks](https://github.com/fountainstudios/AICustomBlocks)
